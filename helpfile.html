Voronoi Sketch Generator

Fusion 360 add-in that lets the user generate Voronoi sketches.

Getting Started| Install/Uninstall| Additional Information| Contact| Version History

Getting Started

This add-in allows the user to generate Voronoi sketches by setting a couple of parameters that will affect the generated forms.

To use the add-in, click the following command

Voronoi Sketch Generator
Runs the command that will pop up a dialog where the parameters for generating a Voronoi sketch can be set. Once created, these sketches may then be used for creating or modifying models.

Usage

Once installed, the "Voronoi Sketch Generator" add-in will create a new button in the menu of the "Model" workspace's "Create" panel.

When running the command these options will be available:

  • Edge Style : Straight or Curved edges
  • Number of Cells (2-256) : Number of cells created in the sketch
  • Pattern width : Width of the area filled with cells
  • Pattern height : Height of the area filled with cells
  • % to scale cells (10-100) : Percentage to scale down each of the cells

Once the command finished we'll get something like this.

These then can be used to create or modify models.

Installation and Uninstallation

You can unload the add-in without uninstalling it by clicking the "Stop" button when the add-in is selected in the list box on the "Add-Ins" tab of the "Scripts and Add-Ins" dialog.

Unchecking "Run on Startup" causes the add-in not to be loaded in future sessions of Fusion 360.
